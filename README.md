# Flipper The Bot
Telegram bot for flipping coins. You can find it [@FlipperTheBot](https://t.me/FlipperTheBot).

## Usage

| Command | Action                              |
|---------|-------------------------------------|
| /start  | Introduces the bot and its commands |
| /flip   | Flips a coin                        |
